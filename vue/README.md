# KSCN 知识共享与协作网络

## 项目概述

KSCN是一个现代化的知识共享平台，提供文件管理、在线服务、用户管理等功能。项目采用Vue.js + Element UI构建，具有响应式设计和优秀的用户体验。

## 已完成功能

### 1. 页面系统
- ✅ **首页** - 重新设计的现代化首页，包含核心功能展示、快捷导航、系统通知等
- ✅ **资源页面** - 公共资源库，文件管理器风格，支持搜索、上传、文件夹管理
- ✅ **云盘页面** - 个人文件库，带存储空间管理，支持文件分享和权限控制
- ✅ **服务页面** - 在线服务展示，包含各种工具和服务的分类介绍
- ✅ **用户中心** - 个人信息管理，支持基本信息修改、密码修改、邮箱验证等
- ✅ **关于我们** - 项目介绍和使命愿景
- ✅ **核心人员** - 开发团队展示
- ✅ **赞助支持** - 赞助方式和权益说明
- ✅ **注意事项** - 平台使用规则和注意事项

### 2. 用户系统
- ✅ **登录页面** - 恢复原始样式，支持用户名/邮箱登录
- ✅ **注册页面** - 恢复原始样式，支持邮箱验证码注册
- ✅ **用户认证** - 基于Token的身份验证系统
- ✅ **权限控制** - 不同页面的访问权限管理

### 3. 导航系统
- ✅ **导航栏** - 响应式设计，支持移动端折叠菜单
- ✅ **用户下拉菜单** - 优化后的用户中心入口和注销功能
- ✅ **面包屑导航** - 文件管理器的路径导航

### 4. 文件管理
- ✅ **文件浏览器** - 支持网格和列表两种视图模式
- ✅ **路径树** - 左侧目录结构导航
- ✅ **文件操作** - 上传、下载、删除、新建文件夹等
- ✅ **搜索功能** - 文件搜索和过滤
- ✅ **存储管理** - 云盘空间使用情况显示

### 5. 响应式设计
- ✅ **大屏幕优化** - 27寸及以上屏幕的舒适布局
- ✅ **中等屏幕** - 1200px-1920px的适配
- ✅ **平板设备** - 768px-1199px的布局优化
- ✅ **移动设备** - 480px-767px的移动端适配
- ✅ **小屏手机** - 480px以下的优化布局

## 技术特点

### 前端技术栈
- **Vue.js 2.x** - 渐进式JavaScript框架
- **Element UI** - 基于Vue的组件库
- **Vue Router** - 官方路由管理器
- **Vuex** - 状态管理模式
- **Axios** - HTTP客户端

### 样式系统
- **CSS Grid & Flexbox** - 现代布局技术
- **CSS变量** - 主题色彩管理
- **媒体查询** - 响应式断点设计
- **模块化CSS** - 按页面分离的样式文件

### 设计理念
- **简洁风格** - 黑白灰为主色调，避免过度装饰
- **用户体验** - 直观的操作流程和反馈机制
- **性能优化** - 懒加载和组件化设计
- **可维护性** - 清晰的代码结构和注释

## 项目结构

```
src/
├── api/                 # API接口
├── assets/             # 静态资源
│   ├── images/         # 图片资源
│   └── styles/         # 样式文件
│       ├── _common.css     # 通用样式
│       ├── home.css        # 首页样式
│       ├── navbar.css      # 导航栏样式
│       ├── footer.css      # 页脚样式
│       ├── information.css # 资源页面样式
│       ├── encyclopedia.css # 云盘页面样式
│       ├── service.css     # 服务页面样式
│       └── profile.css     # 用户中心样式
├── components/          # 公共组件
│   ├── Navbar.vue      # 导航栏组件
│   └── Footer.vue      # 页脚组件
├── router/             # 路由配置
├── store/              # 状态管理
├── utils/              # 工具函数
└── views/              # 页面组件
    └── system/         # 系统页面
        ├── home/        # 首页
        ├── auth/        # 认证页面
        ├── information/ # 资源页面
        ├── encyclopedia/ # 云盘页面
        ├── service/     # 服务页面
        ├── profile/     # 用户中心
        ├── about/       # 关于我们
        ├── team/        # 核心人员
        ├── sponsor/     # 赞助支持
        └── notice/      # 注意事项
```
